## 📂 Estrutura do Repositório

O repositório está organizado de forma clara e padronizada através dos seguintes diretórios principais:

* 📂 **`projetos-academicos/`**: Trabalhos, desafios e projetos desenvolvidos para disciplinas do curso.
* 📂 **`projetos-pessoais/`**: Projetos autorais, desafios de código e estudos independentes.
* 📂 **`documentacao/`**: Slides, PDFs e arquivos de planejamento do portfólio.
